# 🚀 PyTest API Automation Framework

This project is a structured API test automation framework built using Python, PyTest, and requests.

It demonstrates how to design scalable, reusable test automation for backend/API validation.

---

## 🔧 Tech Stack

- Python  
- PyTest  
- Requests  
- PyTest HTML Reports  

---

## 📂 Project Structure
tests/ → test cases

utils/ → reusable helper functions

data/ → test data (future use)

reports/ → test execution reports


---

## ✅ Test Coverage

- Validate API status codes  
- Validate response structure  
- Validate key fields in response  
- Basic data integrity checks  

---

## 🎯 Purpose

This project demonstrates:

- API test automation design
- Reusable test structure
- Backend validation using Python
- Real-world testing approach for REST APIs
